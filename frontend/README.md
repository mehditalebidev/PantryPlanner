# Frontend Workspace

This directory is reserved for the PantryPlanner frontend application.

## Current State

- The frontend scaffold has not been initialized yet.
- `docs/` is for frontend-specific planning and implementation notes.
- `src/` currently exists only as a placeholder entrypoint.

## Planned Responsibilities

- authentication shell and session handling
- recipe creation and import flows
- recipe detail pages with ingredients, steps, and media
- meal-plan views for custom meal slots and flexible date ranges
- grocery-list views derived from selected plans

## Planned Commands

Run from `frontend/` once the scaffold exists:

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run typecheck`
- `npm run test`
