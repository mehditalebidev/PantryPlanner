# UI And Components

## Shared Components

- app shell and navigation
- form field primitives
- inline validation and error states
- empty states and loading states
- media preview components

## Recipe Components

- recipe form
- ingredient line editor
- ordered step editor
- recipe card and recipe detail sections
- media gallery and upload placeholder components

## Meal Plan Components

- plan range header
- meal-slot column or row layout
- planned recipe picker
- slot editor for default and custom sections

## Grocery Components

- generated grocery list
- item checkoff row
- grouped ingredient sections if grouping is added later

## UX Notes

- recipe creation should feel structured but fast
- meal planning should make repeated weekly patterns easy to scan
- grocery lists should be readable on mobile while shopping
